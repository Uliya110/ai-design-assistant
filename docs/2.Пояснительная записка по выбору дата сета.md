# Пояснительная записка по выбору датасета

## 1. Название и источник датасета

**Sentiment Analysis Dataset (Customer Reviews Dataset)**  
**Источник:** платформа Kaggle

---

## 2. Описание датасета

Данный датасет содержит текстовые отзывы пользователей, а также их эмоциональную окраску (sentiment) и числовую оценку.  

Каждая строка представляет собой отдельный отзыв клиента, а столбцы включают:

- `cleaned_review` — текст отзыва пользователя (очищенный)  
- `sentiments` — тональность отзыва: `positive / neutral / negative`  
- `cleaned_review_length` — длина текста отзыва  
- `review_score` — числовая оценка от 1 до 5  

Таким образом, датасет сочетает текстовые и числовые данные, что удобно для анализа поведения клиентов и их восприятия услуги.

---

## 3. Страна и контекст

- Датасет представляет собой обобщённые отзывы пользователей, **не привязан к конкретной стране**.  
- Типичные данные для онлайн-платформ и сервисов:  
  - анализ клиентского опыта  
  - оценка удовлетворённости  
  - изучение эмоциональной реакции пользователей

---

## 4. Период данных

- В датасете отсутствует явная временная метка (дата или период сбора отзывов).  
- Обычно такие наборы формируются на основе накопленных отзывов за длительный период.  
- Используются в задачах анализа текста и машинного обучения.

---

## 📊 Ключевые столбцы

| Столбец                | Описание |
|------------------------|----------|
| `sentiments`           | Эмоциональная окраска: `positive / neutral / negative` |
| `cleaned_review`       | Текст отзыва пользователя |
| `cleaned_review_length`| Длина текста отзыва |
| `review_score`         | Числовая оценка (1–5) |

---

## 🔎 Что показывают данные

- **Эмоциональное состояние клиента** — понять позитив или негатив  
- **Содержание обратной связи** — что понравилось или не устроило  
- **Уровень удовлетворённости** — числовая характеристика опыта  
- **Связь текста и эмоций** — анализ формулировок и тональности  

---

## 📌 Почему выбран этот датасет

Для дипломного проекта «AI-ассистент для 3D-дизайнера»:

- содержит реальные текстовые данные клиентов  
- позволяет анализировать эмоциональную окраску сообщений  
- помогает выявлять удовлетворённость и проблемы клиентов  
- подходит для демонстрации методов анализа текста (NLP)  
- позволяет смоделировать поведение клиента в процессе общения  

Несмотря на отсутствие прямой связи с 3D-дизайном, датасет отражает универсальные принципы взаимодействия с клиентами.

---

## 📌 Связь с дипломной работой

Данный датасет используется для моделирования AI-ассистента, который:

- анализирует текст сообщений клиента  
- определяет его настроение (позитивное / негативное)  
- выявляет потенциальные проблемы  
- помогает дизайнеру быстрее реагировать на запросы  

Особенно важно для 3D-дизайнера, где коммуникация с клиентом напрямую влияет на результат проекта.

---

## 📌 Практическая ценность

Использование датасета позволяет:

- автоматизировать анализ отзывов и сообщений клиентов  
- быстрее выявлять недовольных клиентов  
- улучшать качество коммуникации  
- снижать количество ошибок при работе с заказчиком  
- повышать уровень удовлетворённости клиентов

---

## 📌 Вывод

Выбранный датасет сочетает **текстовые и числовые характеристики**, что позволяет проводить как качественный, так и количественный анализ.  

Он демонстрирует применение методов анализа текста и подготовки данных для создания AI-ассистента.  

Результаты анализа могут быть использованы для улучшения взаимодействия между дизайнером и клиентом, что является ключевой задачей дипломного проекта.
