# 🎨 AI-ассистент для 3D-дизайнера  
## Поиск клиентов и ведение переписки

## 📌 Описание проекта

Данный проект представляет собой AI-ассистента, который помогает 3D-дизайнерам:

- находить потенциальных клиентов  
- анализировать их активность и поведение  
- автоматизировать переписку  
- снижать количество правок и недопониманий  

Ассистент ориентирован на решение реальных проблем дизайнеров:  
непонятные ТЗ, постоянные правки, переговоры о цене и сроки выполнения.

---

## 🎯 Цель проекта

Создать инструмент, который:

- помогает дизайнеру быстрее обрабатывать запросы клиентов  
- анализирует поведение клиентов на основе данных  
- предлагает оптимальные ответы в переписке  
- снижает нагрузку на коммуникацию  

---

## 👤 Целевая аудитория

- 3D-дизайнеры (интерьеры, экстерьеры, предметка)  
- фрилансеры  
- студии визуализации  
- начинающие дизайнеры  

---

## ⚙️ Функционал (MVP)

### 🔍 1. Анализ клиентов
- оценка активности клиента  
- выявление «сложных» клиентов (много правок, частые обращения)  
- сегментация клиентов  

### 💬 2. AI-помощь в переписке
- генерация ответов клиенту  
- объяснение стоимости работы  
- ответы на вопросы о сроках и правках  

### 📊 3. Анализ данных
- обработка датасета Customer Churn  
- поиск закономерностей в поведении клиентов  
- визуализация данных  

---

## 📊 Используемые данные

Источник: Kaggle  
Датасет: Customer Churn (test.csv)

Содержит:
- активность клиентов (минуты звонков)  
- стоимость услуг  
- обращения в поддержку  

📌 В рамках проекта эти данные используются как аналог поведения клиентов дизайнера.

---

## 🧠 Применение анализа данных

В проекте используются:

- `pandas` — обработка данных  
- `matplotlib` — визуализация  
- агрегатные функции (`agg`, `groupby`)  
- поиск максимумов (`max`, `idxmax`)  

Анализ позволяет:
- выделить активных клиентов  
- определить проблемные зоны  
- подготовить основу для AI-логики

## 💡 Проблемы, которые решает проект

- слишком много правок  
- недопонимание с клиентами  
- сложности с ценообразованием  
- потеря времени на переписку  
- стресс от общения  

---

## 🔗 Пример применения

AI-ассистент может:

- объяснить клиенту, почему закатное освещение даёт жёлтый оттенок  
- аргументировать стоимость рендера  
- предложить корректные сроки выполнения  
- фильтровать «невыгодных» клиентов
