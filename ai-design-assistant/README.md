# AI Design Assistant (RAG)

AI-ассистент для дизайнеров, который отвечает на вопросы на основе базы знаний дизайн-гайдлайнов и правил бренда.

Проект использует подход **RAG (Retrieval Augmented Generation)** — поиск информации в базе знаний перед генерацией ответа.

---

## Возможности

* поиск информации в дизайн-гайдлайнах
* ответы на вопросы о правилах дизайна
* рекомендации по оформлению интерфейсов
* работа с текстовой базой знаний (.md файлы)

---

## Структура проекта

```
ai-design-assistant/

src/
    core/        # логика RAG
    parsers/     # чтение документов
    storage/     # работа с векторной базой
    cli/         # запуск программы

data/            # база знаний
notebooks/       # эксперименты
tests/           # тесты
docs/            # документация
configs/         # настройки проекта
```

---

## Технологии

* Python
* LangChain
* FAISS или Chroma (векторная база)
* OpenAI API
* Jupyter Notebook

---

## Установка

Клонировать репозиторий:

```
git clone https://github.com/username/ai-design-assistant.git
cd ai-design-assistant
```

Установить зависимости:

```
pip install -r requirements.txt
```

---

## Запуск проекта

Запуск CLI-ассистента:

```
python -m src.cli.app demo
```

После запуска можно задать вопрос, например:

```
Какой минимальный отступ используется в интерфейсе?
```

---

## Пример базы знаний

Файлы в папке `data/` могут содержать правила дизайна:

```
Минимальный отступ между элементами — 16px
Основной шрифт — Inter
Использовать не более 3 основных цветов
```

AI-ассистент будет отвечать на основе этих документов.
